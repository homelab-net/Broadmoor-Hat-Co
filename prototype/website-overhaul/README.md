# Website Overhaul Prototype (Phase 1)

This prototype demonstrates the broader website overhaul outside of Hat Finder 2.0.

## Included sections
- Conversion-oriented hero with CTA hierarchy:
  - Book Hat Appointment
  - Book Event
  - Shop Ready-to-Wear
- Intent-based top navigation (Shop, Custom Hats, Events, Appointments, About, FAQ, Contact).
- "Choose Your Experience" cards.
- Event package comparison table and inquiry CTA.
- Appointment funnel summary with estimate and handoff to Hat Finder 2.0.
- FAQ accordions and contact/lead form section.
- Mobile sticky CTA bar (Book / Call / Directions).

## Run locally
From repository root:

```bash
python -m http.server 8000
```

Open:

- Full launcher: http://localhost:8000/prototype/full-website-preview.html
- Overhaul page: http://localhost:8000/prototype/website-overhaul/
- Hat Finder: http://localhost:8000/prototype/hat-finder-2.0/
