# Broadmoor Hat Co Prototypes

## Available prototypes
- Full website preview launcher:
  - `prototype/full-website-preview.html`
- Website overhaul prototype (Phase 1):
  - `prototype/website-overhaul/`
- Hat Finder 2.0 prototype:
  - `prototype/hat-finder-2.0/`

## Run locally
```bash
python -m http.server 8000
```

Then open:
- http://localhost:8000/prototype/full-website-preview.html
- http://localhost:8000/prototype/website-overhaul/
- http://localhost:8000/prototype/hat-finder-2.0/
